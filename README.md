## Setup
**Root level**
      
    allprojects {
	repositories {
		...
		maven { url 'https://jitpack.io' }
		}
    }

**Gradle**

    dependencies {
       implementation 'com.github.GakkMedia:MYBLShadhinSDK-android:1.0.0'
    }
    
 ## Usage

More usages about ShadhinSDK, please see the sample.

Open HomeFragment

     ShadhinMusicSdkCore.getHomeFragment()
